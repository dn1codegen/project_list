# Project Management System

Rails 8 приложение для управления проектами с клиентской загрузкой файлов и веб-интерфейсом.

## Особенности

- 🎨 **Красивый веб-интерфейс** для просмотра проектов
- 🔐 **Безопасный доступ** через токены для веб-просмотра
- 📊 **Статистика** по просмотрам проектов для каждого пользователя
- 📱 **Адаптивный дизайн** для мобильных устройств
- 🔑 **Управление ключами** через админ-панель
- 📤 **API** для загрузки проектов клиентским скриптом
- 🖼️ **Галерея изображений** с модальным окном для просмотра
- 📄 **Поддержка PDF** документов с описаниями

## Стек технологий

- Ruby on Rails 8
- Ruby 3.4.8
- SQLite3
- Active Storage
- CSS (кастомные стили)

## Быстрый старт

### 1. Установка зависимостей

```bash
bundle install
```

### 2. Настройка базы данных

```bash
bin/rails db:setup
```

### 3. Создание администратора

```bash
rails runner "Admin.create!(email: 'admin@example.com', password: 'admin123', password_confirmation: 'admin123')"
```

### 4. Запуск сервера

```bash
bin/rails server
```

## Доступ к системе

### Администратор

- **Вход:** `http://localhost:3000/admin/login`
- **Учетные данные:** admin@example.com / admin123
- **Функции:** управление ключами, просмотр статистики, настройка профиля

### Пользователи

Пользователи получают доступ через уникальные токены:

- **Список проектов:** `http://localhost:3000/projects?token=ВАШ_ТОКЕН`
- **Детали проекта:** `http://localhost:3000/projects/1?token=ВАШ_ТОКЕН`

## Основные возможности

### Для администраторов

- ✅ Создание и управление токенами доступа
- ✅ Просмотр статистики по каждому пользователю
- ✅ Отключение/активация токенов
- ✅ Смена email и пароля
- ✅ Фильтрация проектов

### Для пользователей

- ✅ Просмотр списка проектов
- ✅ Поиск по названию, заказчику, адресу
- ✅ Детальный просмотр проектов
- ✅ Просмотр PDF файлов
- ✅ Галерея изображений (замеры, примеры)

## Структура проекта

```
app/
├── controllers/
│   ├── admin_controller.rb          # Управление админкой
│   ├── api_keys_controller.rb       # Управление API ключами
│   ├── link_stats_controller.rb     # Статистика по ссылкам
│   └── projects_controller.rb       # Управление проектами
├── models/
│   ├── admin.rb                     # Модель администратора
│   ├── project.rb                   # Модель проекта
│   ├── project_file.rb              # Файлы проекта
│   ├── project_key.rb               # API ключи для отправки
│   ├── project_view.rb              # Просмотры проектов
│   ├── web_access_key.rb            # Ключи для веб-доступа
│   └── access_log.rb                # Логи доступа
└── views/
    ├── admin/                       # Представления админки
    ├── projects/                    # Представления проектов
    └── link_stats/                  # Представления статистики

client/
├── send.rb                          # Скрипт отправки проектов
├── project_config.yaml             # Конфигурация клиента
└── README.md                        # Документация клиента
```
