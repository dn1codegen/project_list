# Информация об API ключах

## Два типа API ключей

Система поддерживает два типа API ключей для разных целей:

### 1. Ключ для отправки проектов (API)

**Назначение:** Отправка проектов через клиентский скрипт (send.rb)

**Где использовать:**
- В файле `client/project_config.yaml`
- В параметре `api_key` при вызове скрипта

**Пример:**
```bash
ruby send.rb project_config.yaml
```

### 2. Ключ для веб-доступа (по ссылке)

**Назначение:** Просмотр проектов через веб-интерфейс по ссылке

**Где использовать:**
- В URL как параметр `?token=...`
- При открытии проектов в браузере

**Пример:**
```
http://localhost:3000/projects/1?token=ВАШ_КЛЮЧ
```

## Создание ключей через веб-интерфейс

### Шаги:

1. **Откройте страницу управления ключами:**
   ```
   http://localhost:3000/api_keys?token=ВАШ_WEB_ACCESS_KEY
   ```

2. **Выберите тип ключа:**
   - 🔑 **Для отправки проектов** - используется в клиентском скрипте
   - 🌐 **Для веб-доступа** - используется в браузере

3. **Введите название ключа** (например: "API для проекта 1")

4. **Нажмите "Создать ключ"**

### Типы ключей:

| Тип | Использование | Где применять |
|-----|---------------|----------------|
| 🔑 Отправка | Скрипт send.rb | `client/project_config.yaml` |
| 🌐 Веб-доступ | Браузер | URL `?token=...` |

## Примеры использования

### Отправка проекта через API

```bash
cd client
ruby send.rb project_config.yaml
```

### Просмотр проекта по ссылке

Скопируйте ключ для веб-доступа и добавьте его к URL проекта:

```
http://localhost:3000/projects/KM-0001?token=e51c0daf5a08e3c74ac914b334972a16873f35dca73ebf046f533b2493306714
```

Или используйте поисковую форму с параметром `?token=...`:

```
http://localhost:3000/projects?token=e51c0daf5a08e3c74ac914b334972a16873f35dca73ebf046f533b2493306714
```

## Управление API ключами

### Создание нового ключа

**Через веб-интерфейс:**
```
http://localhost:3000/api_keys?token=ВАШ_WEB_ACCESS_KEY
```

**Через консоль:**
```bash
rails runner "ApiKey.create!(name: 'Новый ключ', key_type: 'web_access')"
```

### Просмотр всех ключей

```bash
rails runner "ApiKey.all.each { |k| puts \"Имя: #{k.name}, Тип: #{k.key_type_name}, Ключ: #{k.key[0..30]}...\" }"
```

### Активация/деактивация ключей

```bash
# Активировать все ключи
rails runner "ApiKey.update_all(active: true)"

# Деактивировать все ключи
rails runner "ApiKey.update_all(active: false)"

# Активировать конкретный тип
rails runner "ApiKey.where(key_type: 'web_access').update_all(active: true)"
```

## Безопасность

- ⚠️ **НЕ ИСПОЛЬЗУЙТЕ** ключ для отправки для веб-доступа
- 📋 Храните ключи в безопасном месте
- 🔐 Регулярно создавайте новые ключи
- 🚫 Не публикуйте ключи в открытых репозиториях
- 🔄 Меняйте ключи при подозрении на утечку
- 🔑 Используйте разные ключи для разных проектов
